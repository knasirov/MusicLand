# API Endpoints

## HTML API

### Root

- `GET /` - loads React web app

## JSON API

### Users

- `POST  /api/users`
- `GET   /api/users/:id` - returns user page
- `PATCH /api/users/:id`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Tracks

- `GET /api/tracks` - all tracks for stream page
- `POST /api/tracks`
- `GET /api/tracks/:id`
- `PATCH /api/tracks/:id`
- `DELETE /api/tracks/:id`

### Comments

- `POST /api/comments`
- `DELETE /api/comments/:id`
